# Nuxt 3

This is an example of how to use the [Rick and Morty API](https://rickandmortyapi.com/) with Nuxt 3.

## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## Development Server

Start the development server on `http://localhost:3000`

```bash
npm run dev
```
